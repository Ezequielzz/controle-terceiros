# Sistema Kanban para Controle de Usinagem de Terceiros

## Descrição

O Sistema Kanban para Controle de Usinagem de Terceiros tem como objetivo auxiliar no acompanhamento e organização de peças mecânicas enviadas para processos externos de usinagem.

A aplicação foi concebida para substituir o controle realizado anteriormente em quadros Kanban digitais, oferecendo uma solução simples, intuitiva e focada nas necessidades do processo produtivo do cliente.

Por meio de cartões organizados em colunas que representam as etapas do fluxo de trabalho, o usuário poderá visualizar rapidamente o status de cada serviço, acompanhar prazos, localizar Ordens de Serviço (OS) específicas e registrar informações relevantes para o acompanhamento das peças.

O sistema foi projetado para operar localmente, sem necessidade de conexão com a internet, priorizando simplicidade, baixo custo de manutenção e facilidade de utilização no ambiente de trabalho.

## Objetivos

* Organizar Ordens de Serviço em um fluxo visual do tipo Kanban.
* Facilitar o acompanhamento dos serviços enviados para terceiros.
* Permitir a consulta rápida de informações por número da OS.
* Reduzir o risco de perda de informações operacionais.
* Centralizar os dados necessários para acompanhamento da produção.
* Disponibilizar uma ferramenta simples e intuitiva para uso diário.

## Funcionalidades Principais

* Criação de cartões de acompanhamento.
* Edição de informações dos cartões.
* Movimentação de cartões entre colunas.
* Pesquisa por número da Ordem de Serviço.
* Persistência local dos dados.
* Exportação e importação de backups.
* Destaque visual para acompanhamento de prazos.
